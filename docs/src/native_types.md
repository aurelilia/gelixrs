# Builtin Types

This page lists all types currently part of the gelix spec.

Name | Type | Literal | Byte Size
--- | --- | --- | ---
`None` | singleton | --- | 0
`bool` | boolean | `true; false` | 1
`i8` | signed int | `14i8` | 1
`i16` | signed int | `14i16` | 2
`i32` | signed int | `14i32` | 4
`i64` | signed int | `14` | 8
`f32` | float | `32.0f32` | 4
`f64` | float | `32.0` | 8
